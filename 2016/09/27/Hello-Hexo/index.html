hexo&github 搭建博客

HTML 书籍
  • CSS
  • HTML 标签
  • javascript 标签

今天折腾了一天终于把hexo博客搭建好了，下面就我搭建过程做个总结。

搭建环境：

  • windows 7 32
  • node.js 4.4.4 http://nodejs.cn/
  • npm 2.15.1
  • git客户端 https://git-for-windows.github.io/

    步骤一：在github上搭建仓库

    仓库的名称要用这个格式：username.github.io

步骤二：node,git，hexo安装

node.js直接下载安装，安装完成后进入命令行输入：

  • node -v 能够正常返回版本号的说明安装成功
  • npm -v 能够正常返回版本号的说明安装成功
  • node.js里面集成了npm所以，不用再单独安装

git安装下载后，直接运行安装，安装成功后会有三个应用程序分别是

  • git-bash
  • Git Shell
  • GitHub

安装hexo
在本地任意盘下新建一个文件夹hexo,然后右键选择git bash here或者直接运行git bash进入到刚才创建的hexo文件

npm install hexo -g   # -g表示全局安装, npm默认为当前项目安装
hexo init hexo        # 执行init命令初始化到你指定的hexo目录
npm install           # install before start blogging
hexo generate         # 自动根据当前目录下文件,生成静态网页
hexo server           # 运行本地服务

好了，在浏览器输入: http://localhost:4000，就可以看到landscape主题的hexo页面了。

关于如何更换主题以及对hexo博客页面的布局修改可以到这里进一步了解https://github.com/hexojs/hexo/wiki/Themes

hexo常用命令

hexo n "我的博客" == hexo new "我的博客" #新建文章 
hexo p == hexo publish 
hexo g == hexo generate#生成 
hexo s == hexo server #启动服务预览 
hexo d == hexo deploy#部署

服务器

hexo server #Hexo 会监视文件变动并自动更新，您无须重启服务器。 
hexo server -s #静态模式 
hexo server -p 5000 #更改端口 
hexo server -i 192.168.1.1 #自定义 IP

hexo clean #清除缓存 网页正常情况下可以忽略此条命令 
hexo g #生成静态网页 

部署到github上之前先要安装hexo-deployer-git
npm install hexo-deployer-git --save
hexo d #开始部署

以上就是我部署hexo的整个过程，很多其它情况（看其它教程中有不同的错误出现）我在部署过程没有遇到，希望我的部署过程能给你带去些许帮助，也希望能和大家一起交流更多的话题^_^包括但不限于前端，github，木吉他，篮球，家具设计等等
